Shortcuts

Project

Ctrl + N New project
Ctrl + O Open existing project
Ctrl + S Save project
Ctrl + F4 Close project
Ctrl + M Create script from model

Geometry

N Work with Nodes
E Work with Edges
L Work with Labels
Alt + N Add Node
Alt + E Add Edge
Alt + L Add Label
Space Open properties dialog

Others

F1 Switch to Welcome page
F2 Switch to Problem
F3 Switch to Results (when is active)
F4 Switch to Study
F5 Switch to Optilab (when is active)
F11 Full screen mode (open/close)

Controls

Ctrl + Left click Add node/edge/label directly by mouse
Ctrl + Shift + Left click Move nodes/edges/labels by mouse
Middle button double click or Shift + Left double click Zoom to fit
Mouse wheel Zoom the workspace
Shift + Left click or Middle button click Pan over the workspace (you can also use arrows keys)
