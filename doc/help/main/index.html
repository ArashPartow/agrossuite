Agros 2D

hp-FEM multiphysics solver based on Hermes2D library

Agros 2D is a multiplatform application for the solution of physical problems in two dimensions (planar and axisymmetric arrangement). It includes process of modeling, solving and postprocessing of the model. Allows to solve:

  • electrostatic field,
  • electric current field,
  • magnetostatic field,
  • harmonic magnetic field,
  • heat transfer.

Key features

  • Interactive creation of geometry and AutoCAD DXF (Drawing Exchange Format) import.
  • Visualization of field variables.
  • Chart, data and images export.
  • Local values, surface and volume integrals.
  • Scripting support.
