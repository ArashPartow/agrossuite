Agros 2D

hp-FEM multiphysics solver based on Hermes2D library

Key features

  • interactive creation of geometry + dxf import
  • visualization of field variables
  • chart and data export
  • local values, surface and volume integrals
  • scripting support

Supported physical fields

  • electrostatic
  • magnetostatic
  • heat transfer
  • electric current field

Download Agros 2D

Requirements

  • Hermes2d 1.0
  • Nokia Qt 4.5 (http://www.qtsoftware.com/products/)
  • Qwt 5.0 (http://qwt.sourceforge.net/)

Binaries

  • Ubuntu 32 bit: 
    libhermes2d_1.0-1ubuntu2_i386.deb
    agros2d_0.7-1ubuntu1_i386.deb
  • Windows 
    Agros2D_0.7.0.exe
