Remote Control

Agros2D support remote control. You can use agros2d-remote command for remote controling of Agros2D. For example:

agros2d-remote "openDocument("data/electrostatic_planar_capacitor.a2d");"
agros2d-remote "solve();"
agros2d-remote "result = pointResult(0.12, 0.32);"
agros2d-remote "print(result.V);"

If you can run script file you must using -script switch. For example:

agros2d-remote -script "data/script/electrostatic_axisymmetric_capacitor.qs"

You can use commands described in the section Scripting.
