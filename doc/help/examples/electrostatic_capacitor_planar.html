Nodes

Select "Operate on nodes" button from main toolbar and place 6 points with coordinates [0.1, 0.0], [0.2, 0.0], [0.1, 0.2], [0.0, 0.2], [0.0, 0.3] and [0.1, 0.0]. You can move mouse to requested place and press left button together with control key, or by pressing <Alt>+n and entering the point coordinates.

Edges

Select "Operate on edges" button from main toolbar and draw edges. Press <Control> key and select in sequence start and end node of the line.

Labels

Select "Operate on labels" button and place block label into inner area of the capacitor. To place label you can use left mouse button together with control key or <Alt>+l shortcut.
