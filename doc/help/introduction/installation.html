Installation

Compilation on Debian-based systems

This procedure is tested on Ubuntu 9.04 "Jaunty Jackalope".

Installing packages needed for compilation

apt-get install libjudy-dev freeglut3-dev libsuitesparse-dev
apt-get install libqwt5-qt4-dev

Downloading source codes

Hermes2D

http://hpfem.math.unr.edu/git/gitweb.cgi/hermes2d.git/

Downloading with Git:

git clone git://hpfem.math.unr.edu/git/hermes2d.git

Downloading with GNU Wget:

wget http://hpfem.math.unr.edu/git/gitweb.cgi/hermes2d.git?a=snapshot;h=HEAD;sf=tgz

Agros 2D

http://hpfem.math.unr.edu/git/gitweb.cgi/agros2d.git/

Downloading with Git:

git clone git://hpfem.math.unr.edu/git/agros2d.git

Downloading with GNU Wget:

wget http://hpfem.math.unr.edu/git/gitweb.cgi/agros2d.git?a=snapshot;h=HEAD;sf=tgz
